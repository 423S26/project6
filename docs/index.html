HIPAA Compliant Word Processor for ABA Therapy Use

This is a product website that contains user documentation, developer documentation, and UML diagrams associated with the HIPAA Compliant Word Processor. This project is part of a capstone course ESOF 423.

User Documentation

The word processor creates, edits, formats, stores and prints text based documents. It’s unique because it is secure and HIPAA compliant to prevent leaks of confidential patient information. Our client wants it to make documentation streamlined and secure, the client also wants to share the documents with clients and other individuals related to the patient (e.g. the patient’s parents) since they like seeing the notes our client takes.
